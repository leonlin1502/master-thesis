# Master Thesis

This repository contains the PDF of my master thesis and the code used in the numerical experiment.

The main contribution of this thesis is an algorithm for the perturbation of the SVD problem that combines a Taylor-like perturbation expansion and techniques from the orthogonal Procrustes problem. While both techniques are well known individually, their combination for this problem is, to the best of my knowledge, novel.

The file `master_thesis_code.ipynb` provides the numerical implementation for Subsection 3.1.1 (Numerical Justification) of my master thesis (`master_thesis.pdf`). The core algorithm implemented in the code is proposed in Chapter 3 of the thesis.

## Contents

- `master_thesis.pdf` – Full master thesis document  
- `master_thesis_code.ipynb` – Numerical implementation for the experiment in Subsection 3.1.1
